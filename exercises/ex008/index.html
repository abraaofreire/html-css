Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito, usando a tag <b></b> não semântica.

Nesta frase, temos um termo em destaque, usando a tag <strong></strong> semântica.

Itálico / Ênfase

Nesta frase, temos um termo em itálico, usando a tag <i></i> não semântica.

Nesta frase, temos um termo com ênfase, usando a tag <em></em> semântica.

Texto marcado

Podemos criar também um texto marcado, usando a tag <mark></mark> não semântica.

E no outro parágrafo, temos outro texto marcado.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido, usando a tag <del></del> semântica. Para indicar que o texto deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido, usando a tag <ins></ins> semântica. Para dar um ênfase e indicar que ele foi adicionado.

Existe também o sublinhado com a tag <u></u>, não semântica.

Texto sobrescrito

Para inserir coisas do tipo: x4+3. Usamos a tag <sup></sup> semântica.

Texto subscrito

Para inserir coisas do tipo: H2O. Usamos a tag <sub></sub>.
